FEATURES

LiveLog でできること

ライブ好きが本当にほしかった機能を、ぎゅっと詰め込みました。

ライブを記録・管理

予定・参加済み・不参加をまとめて管理。年別に振り返れて、自分だけのライブ史ができあがる。

チケット状況を見える化

抽選受付中・当選・支払い待ち・発券待ち…細かいステータスで「次に何をすべきか」が一目でわかる。

期限・開演時間を管理

申込締切や開場・開演時間を登録。Googleカレンダーにも追加できて、うっかりを防止。

セットリストを残す

手動でも、setlist.fm や LiveFans から自動取得でも。あの日の曲順を、ずっと残せる。

メモ・感想を保存

自分だけのメモと、シェア用の感想を分けて記録。熱が冷めないうちに気持ちを残そう。

友達と共有

公開プロフィールで、参加履歴やセトリ・感想をシェア。友達のライブ遍歴を覗くのも楽しい。

入力はもう、AIにおまかせ。

チケット画面のスクショや、e+ のテキストを貼るだけ。AIが公演名・日付・会場をぜんぶ読み取って、自動で登録します。面倒な手入力とはお別れ。
