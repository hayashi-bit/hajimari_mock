## 試してほしいこと（5つ）

### 1. まず、思っていることを何でも話してみる
- 仕事のこと、やりたいこと、モヤモヤしていること、何でもOK
- 「正解」はないので、思いついたまま話してください
- 短い言葉でも、長い文章でも大丈夫

### 2. AIの質問に答え続けてみる
- miiiroはアドバイスしません。質問だけします
- 「答えにくいな」と思ったら、その気持ちをそのまま伝えてOK
- 5往復くらい続けると「今日のまとめを見る」ボタンが出ます

### 3. 「今日のまとめ」を見てみる
- 5往復以上会話したら、画面下に「今日のまとめを見る」ボタンが表示されます
- タップすると、会話の振り返りが表示されます
- 「自分が話したことがちゃんと整理されているか」を確認してみてください

### 4. 翌日以降にもう一度開いてみる
- 同じスマホ・同じブラウザで開くと、前回の記憶が引き継がれます
- AIが前回の内容を踏まえて問いかけてくれるか確認してください

### 5. 履歴を見てみる
- ヘッダー左の「履歴」をタップすると、過去のセッション一覧が見れます
- 過去のセッションをタップすると、続きから会話できます

## フィードバックで教えてほしいこと

### 体験について
- 「話しやすかった」or「話しにくかった」、その理由
- AIの質問は「ちょうどいい」or「多すぎる」or「的外れ」
- 「自分で考えた感覚」が残ったか、それとも「AIに聞かれただけ」だったか
- ChatGPTと比べてどう感じたか

### 操作について
- 迷った場所、分からなかった場所
- 「こうなっていたらよかった」と思ったこと
- 音声入力は使ったか、使いやすかったか

### 継続意向
- 「また使いたい」と思ったか
- もし月額1,100円だったら、使い続けるか
- どんな場面で使いたいと思ったか

## 注意事項
- **同じスマホ・同じブラウザ**で開いてください（履歴が引き継がれます）
- シークレットモード（プライベートブラウズ）では履歴が消えます
- 端末を変えると別の人として扱われます
- 会話内容は開発チームが分析のために確認する場合があります
