# サービス選定の基本方針（クロコ・マナス共通ルール）

作成: 2026-06-18 ／ 林の明示指示によるルール。クロコ（Claude）もマナス（実装AI）も従う。

## ルール
**外部サービス（アー写・メタデータ取得APIなど）を評価・採用するときは、必ず「林が実際に参戦したアーティストの実リスト」でカバー率を検証してから決める。**

- このサービスのコアユーザーは**現場好き＝マイナー/インディーのライブに通う人**。
- だから**メジャー中心の一般論や机上比較で決めない**。必ず**実リスト**でヒット率を確認する。
- 採否の基準は「**前髪ぱっつん少年のような小規模アーティストまで取れるか**」。
  コアユーザーが見るマイナーが取れてこそ価値。

**結論：Spotify と Apple Music は実リストをほぼ100%カバー（前髪ぱっつん少年も取れる）。Deezer(無料)は前髪ぱっつん少年が抜ける。**
→ 無料優先なら Deezer＋ユーザー投稿。最小バンドまで自動で取りたいなら Spotify(Premium) か Apple Music($99/年) が必要。

## 注意（古い情報に注意）
- **Last.fm はアーティスト画像をAPIから削除済み（2019〜）＝全員プレースホルダ**。画像源にならない。
- iTunes Search API（無料）は**アーティスト画像を返さない**（アルバム画像のみ）。
- MusicBrainz は画像を持たず、Wikidata/fanart 経由＝**JPインディーはスカスカ**。
