## 2. 取得する情報
本サービスは、以下の情報を取得します。
- **アカウント情報**：Googleログインを通じて取得するメールアドレス、表示名、プロフィール画像。
- **利用者が入力する情報**：ライブの参戦記録、チケット情報・ステータス、セットリスト、メモ、感想、投稿画像（フライヤー等）。
- **Gmail連携で取得する情報**：利用者が連携を許可した場合に限り、チケットに関連するメールから抽出した公演名・日付・会場・枚数・ステータス等の情報。
- **利用ログ**：アクセス日時、操作ログ等の技術情報。

## 3. Gmailデータの取り扱い（Google API Services User Data Policy / Limited Use 準拠）
- **利用目的は「チケット情報の自動取り込み」のみ**に限定します。
- メールの**本文は保存しません**。抽出に必要な処理を行った後、構造化した結果（公演名・日付等のメタデータ）のみを保存します。
- Gmailから取得した情報を、広告目的で利用・第三者に販売することはありません。
- 本サービスのGmailデータの利用は、Googleの **Limited Use 要件**に準拠します。
- 利用者はいつでもGmail連携を解除できます。

## 4. 利用目的
- 本サービスの提供・維持（記録管理、チケット期限の通知等）。
- 機能改善、不具合対応、サポート対応。
- 不正利用の防止。

## 6. データの保管・セキュリティ
- 個人データはクラウド上に保管し、アクセス制限等の安全管理措置を講じます。

## 7. 退会・データの削除
- 利用者は、本サービス内の導線から、自身のデータの削除および退会を行えます。
- 退会時、当方は法令で保持が必要な場合を除き、利用者の個人データを削除します。

- 利用者は、自身の個人情報の開示・訂正・利用停止等を、上記問い合わせ先に請求できます。
