# Gmail連携：アカウントとメールの基本仕様（確定）＋ Manus指示

作成: 2026-06-21 ／ 林の明示指示による仕様確定。クロコ・マナス共通。

## 確定仕様（明確化）

**1ユーザー ＝ 自分のGmail1本だけ。**
各ユーザーは「自分のLiveLogアカウント」で「自分のGmail」を連携し、「自分のチケットメール」だけを取り込む。

| 項目 | 仕様 |
|---|---|
| 1人が複数のメアドを登録する | **仕様なし**。1ユーザーは自分のGmail1本のみ連携。複数Gmailの一括取り込みは提供しない。 |
| 1人が他人のメアドを見る | **仕様なし**。他人のGmail／チケットメールを自分のアカウントに取り込まない。データは各ユーザーに完全分離。 |
| yukatoのメールを eiji が取り込む | **しない**。yukatoは自分のアカウントで自分のGmailを連携して使う。 |
| PDF「問題4」全アカウント並列スキャン | **実装しない**。複数アカウント前提の修正案であり、本仕様では不要。 |

> 補足：将来「1人が自分のメイン＋サブGmailを両方繋ぎたい」というニーズが出たら、その時に改めて検討する（現時点では不要・実装しない）。
